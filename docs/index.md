# Introduction to neoLAT

**NeoLAT** (Local Agentic AI Testbed) is a comprehensive environment for evaluating local AI agents.

## Goals

- Facilitate comparison of local inference engines.
- test cognitive capabilities on standardized tasks.
- Provide a modular and extensible research platform.

## Key Features

- **Local Inference**: MLX, Ollama support.
- **Cognitive Tasks**: Sudoku, ARC.
- **Analysis**: Integrated with WandB.
